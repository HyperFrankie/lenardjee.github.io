De basis

De basis van deze website is gecreëerd door GitHub. Github is een site waar je makkelijk bestanden kunt sorteren en combineren door een repository aan te maken. Een repository is een project waarin je bestanden kunt aanmaken, bewerken en uploaden. Ook is het mogelijk om hele repositorys te kopiëren.

Als je een repository hebt gemaakt is het tijd om bestanden aan te maken die de site bouwen. Het is hierbij van belang dat je bestanden de goede naam geeft. Er zijn verschillende soorten bestanden die gebruikt kunnen worden om de site te bouwen, namelijk .css bestanden, .js bestanden en .html bestanden.

bestanden in CSS, HTML en JavaScript

Zoals te zien hebben we 2 bestanden gemaakt die eindigen op .css, dat betekend dat het bestand CSS code bevat. Ook is er 1 bestand dat eindigt op .js, wat JavaScript betekend. Als laatste zijn en 6 bestanden die eindigen op .html, wat betekend dat de code in het bestand HTML code is

De 6 bestanden in HTML code maken de pagina's van de website, zo bevat html.html de informatie voor de sub-pagina HTML en css.html de informatie over de sub-pagina CSS.

De 2 bestanden in CSS code bevatten informatie die in de HTML code opgeroepen kan worden. Denk hierbij aan een bepaalde stijl die je kan oproepen waar je hem nodig hebt in de HTML code. Hierover zal meer duidelijk gemaakt worden in de sub-pagina CSS
