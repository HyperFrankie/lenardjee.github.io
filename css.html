CSS, oftewel Cascading Style Sheets, laat men het uiterlijk van de website bepalen. Hiermee kunnen bepaalde klassen worden gedefinieërd, die vervolgens in HTML gebruikt worden door class="test" te gebruiken. Als in CSS dan bepaalde waarden voor de klasse test zijn bepaald, worden deze waarden gebruikt in plaats van de standaardwaarden. De bestandsextensie voor CSS is .css.

Moeilijkheidsgraad

CSS is een van de makkelijkere programmeertalen om te leren. CSS gebruikt bijvoorbeeld weinig 'echte' programmeertaalfuncties, zoals JavaScript wel doet. CSS en JavaScript kunnen wél samenwerken.

Wat is er mogelijk?

Met CSS is het mogelijk om het uiterlijk van alle beschikbare HTML-elementen aan te passen. Ook kunnen er met CSS verschillende klassen worden aangepast en gecreëerd. Met het aanpassen kan er gedacht worden aan de achtergrondkleur, het lettertype en -grootte, randen, animaties (in de meest recente versie) en nog een aantal andere dingen.

Hoe kan een CSS-bestand in een HTML-bestand worden gezet?

Dit door het volgende in te voeren: 


<link rel="stylesheet" type="text/css" href="url_naar_css-bestand">
