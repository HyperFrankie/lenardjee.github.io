CSS, oftewel Cascading Style Sheets, laat men het uiterlijk van de website bepalen. Hiermee kunnen bepaalde klassen worden gedefinieërd, die vervolgens in HTML gebruikt worden door class="test" te gebruiken. Als in CSS dan bepaalde waarden voor de klasse test zijn bepaald, worden deze waarden gebruikt in plaats van de standaardwaarden. De bestandsextensie voor CSS is .css.

Moeilijkheidsgraad

CSS is een van de makkelijkere programmeertalen om te leren. CSS gebruikt bijvoorbeeld weinig 'echte' programmeertaalfuncties, zoals JavaScript wel doet. CSS en JavaScript kunnen wél samenwerken.

Wat is er mogelijk?

Met CSS is het mogelijk om het uiterlijk van veel beschikbare HTML-elementen aan te passen. Ook kunnen er met CSS verschillende klassen worden aangepast en gecreëerd. Met het aanpassen kan er gedacht worden aan de achtergrondkleur, het lettertype en -grootte, randen, animaties (in de meest recente versie) en nog een aantal andere dingen.

Hoe kan een CSS-bestand in een HTML-bestand worden gezet?

Dit kan door het volgende in te voeren: 


<link rel="stylesheet" type="text/css" href="url_naar_css-bestand">

Een CSS-bestand kan er als volgt uitzien:


body /* Elementnaam */ {
    background-color: #FFFFFF; /* Achtergrondkleur */
    font-face: "Trebuchet MS", Helvetica, sans-serif; /* Te gebruiken lettertypen, aangeraden wordt om met het/de gewenste lettertype(n) te beginnen en eindigen met 'sans-serif', 'serif' of 'monospace' */
    color: #252525; /* Tekstkleur */
}

.blauw /* Naam voor een klasse */ {
    background-color: blue; /* Engelse naam voor de kleur #0000FF (blauw) */
    width: 80%; /* Breedte van het element */
    color: #FFFFFF; /* Kleur van de tekst, overschrijft de eerder benoemde #252525 */
    border: 1px solid #FFFFFF; /* Voor een doorlopende rand aan alle kanten van één pixel breed in de kleur wit (#FFFFFF) */
}
