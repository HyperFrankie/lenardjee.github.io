HTML code geeft de opbouw van de site weer zoals wij hem zien. De code werkt met een basis opbouw die elke pagina heeft die gemaakt is met HTML:


<head>
    <title>Paginatitel</title>
</head>
<body>
    <p>Dit is tekst</p>
</body>

Dit is in principe het begin van een website; het gebruiken van een head-element en een body-element. Tussen het head-element kan metadata worden gebruikt - die bepalen bijvoorbeeld wat voor karakterset wordt gebruikt, of, in dit geval, de titel van de pagina die in het tabblad wordt gebruikt. In het body-element wordt de eigenlijke website gemaakt. Hier wordt, in dit geval, het element p gebruikt. Dit staat voor een alinea.

Hieronder staat een voorbeeld van wat er ook in het head-element geplaatst kan worden, in dit geval een link naar een CSS-bestand van deze website. Merk op dat er bij het element link geen sluitende tag (</link>) gebruikt hoeft te worden.


<link rel="stylesheet" type= "text/css" href="https://rawgit.com/DarkShadowTNT/schoolproject/master/resources.css">

Ook kunnen er in head scriptbestanden worden geplaatst, in het geval hieronder naar een JavaScript-bestand van de website. Merk op dat er hier wel een sluitende tag wordt gebruikt (</script>).


<script src="https://rawgit.com/DarkShadowTNT/schoolproject/master/resources.js"></script>
