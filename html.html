HTML code geeft de opbouw van de site weer zoals wij hem zien. De code werkt met een basis opbouw die elke pagina heeft die gemaakt is met HTML:

⁣<head>
⁣ ⁣ <title>Paginatitel</title>
</head>
<body>
⁣ ⁣ <p>Dit is text.</p>
</body>

De indeling van <head> en <body>

Zoals te zien kan het <head> deel gebruikt worden om een titel van de pagina te laten zien helemaal bovenaan, boven de url-balk. Verder kan het <head> deel ook gebruikt worden om allerlei soorten bestanden op te vragen van verschillende plaatsen. Zo hebben wij bijvoorbeeld in onze code dit stukje in her <head> deel:

⁣<link rel="stylesheet" type="text/css" href="https://rawgit.com/DarkShadowTNT/schoolproject/master/resources.css">
