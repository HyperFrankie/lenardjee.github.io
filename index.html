Welkom!

Deze site hebben wij gemaakt om jullie iets bij te brengen van hoe je een site maakt met behulp van GitHub.

GitHub is een site waarbij je projecten aanmaken waarin je bestanden kunt maken. Voor deze site hebben wij twee verschillende projecten aangemaakt, een project om de broncode in te zetten en een project om de site zelf mee te maken.

In GitHub kun je heel makkelijk een site maken door een project aan te maken met als naam: gebruikersnaam van het account + .github.io. Wij hebben dus als projectnaam van het project dat de site maakt lenardjee.github.io.

Wij hebben onze site onderverdeeld in vier sub-pagina's waarin we telkens een apart aspect van de opbouw van de site behandelen. Verder hebben wij ook een sub-pagina met wat informatie over onszelf.

In de sup-pagina basis leggen we uit hoe we GitHub hebben gebruikt om onze bestanden te ordenen en wat algemene informatie over hoe we de site hebben gemaakt.

Verder hebben we drie pagina's waarin we de verschillende soorten code behandelen, in HTML leggen we de HTML code uit die de opbouw van de site regelt, in CSS behandelen we hoe CSS de stijl van de pagina regelt en in JavaScript leggen we uit wat JavaScript toevoegd aan een pagina.
