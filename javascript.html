JavaScript kan gebruikt worden voor de interactie met een website. Vanwege dit - en het feit dat het afstamt van de programmeertaal C - is deze programmeertaal iets lastiger om te leren.

De bestandsextensie voor Javascript is .js

JavaScript-voorbeeld

Het bekende 'Hallo, wereld!' ziet er bij JavaScript het volgt uit:


alert('Hallo, wereld!')

Dit is nog een van de meer simpelere scripts die kunnen worden gemaakt in JavaScript. Dingen kunnen echter snel ingewikkeld worden, zoals hieronder ook te zien. Dit stuk script komt uit het document dat onderdeel is van de navigatiebalk aan de bovenkant van de pagina.


function smallScreens() {
    var x = document.getElementById("mainNavBar");
    if (x.className === "mainNavigationBar") {
        x.className += " responsive";
    } else {
        x.className = "mainNavigationBar"
    }
}

In het voorbeeld hierboven kunnen duidelijk overeenkomsten gezien worden met een andere programmeertaal, namelijk C. Hierdoor is het ook een van de moeilijkere programmeertalen om te leren.

JavaScript-document toevoegen aan HTML-pagina

Om een JavaScript-bestand toe te voegen aan HTML, kan de volgende code gebruikt worden:


<script src="pad_naar_javascript-bestand"></script>

Zo hebben wij bijvoorbeeld deze code om een JavasCript bestand in te voegen:


<script src="https://rawgit.com/DarkShadowTNT/schoolproject/master/resources.js"></script>
