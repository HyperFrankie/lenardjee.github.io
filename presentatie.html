Presentatie-ondersteuning

Voorbeeld van hoe het wél en niet moet: properties

NIET


body {
    #10A90B: background-color;
}


body {
    #10A90B= background-color;
}

In de bovenstaande voorbeelden wordt een eigenschap (#10A90B) vóór een property (background-color) gezet. Dit gaat niet werken.

WEL


body {
    background-color: #10A90B;
}

In het bovenstaande voorbeeld wordt een property (background-color) vóór een eigenschap gezet (#10A90B). Dit werkt wel.

Selectiekleur en het aanmaken van een klasse


.pas-selectie-aan {
  height: 150px;
  width: 150px;
  border: 1px solid black;
  background-color: #FFFFFF;
}

.pas-selectie-aan::selection {
  background: #149CBE;
  color: #FFFFFF;
}

wordt gebruikt in


<div class="pas-selectie-aan">
  <p>Dit is testtekst. Enkel ter demonstratie.</p>
</div>

om vervolgens dit te maken:

Dit is testtekst. Enkel ter demonstratie.

Simpele animaties


.animatie {
  width: 95%;
  height: 25px;
  background-color: #FFFFFF;
  color: #000000;
  transition-duration: 5.0s;
  font-size: 10px;
  border: 1px solid black;
}

.animatie:hover {
  background-color: #000000;
  color: #FFFFFF;
  font-size: 20px;
  border: 1px solid black;
}

wordt gebruikt in


<div class="animatie">
  <p>Het element wordt zwart en de tekst wordt wit en groter als de muis erop geplaatst wordt. Ook wordt de rand wit.</p>
</div>

om vervolgens dit te maken:

Het element wordt zwart en de tekst wordt wit en groter als de muis erop geplaatst wordt. Ook wordt de rand wit.
